# FITMIND AI - SaaS de Gestão e Prescrição Fitness Inteligente

Projeto acadêmico de um software em nuvem (SaaS) voltado para academias e personal trainers, focado na criação ágil de treinos e dietas hiper-personalizados utilizando Inteligência Artificial.

## 🚀 Tecnologias Utilizadas
- HTML5 Semântico e Acessível
- CSS3 Responsivo (Grid e Flexbox) com animações fluidas
- JavaScript Vanilla (Puro) para controle de rotas/abas e simulações
